# Proyecto de Pruebas Unitarias

## Descripción
Este proyecto contiene una clase `ValidadorRegistroTest` que realiza pruebas unitarias para validar el registro de usuarios.

## Archivos
- `ValidadorRegistroTest.java`: Clase que contiene las pruebas unitarias para validar el registro de usuarios.

## Ejecución
Para ejecutar las pruebas unitarias, sigue estos pasos:

1. Abre el proyecto en tu IDE favorito.
2. Localiza el archivo `ValidadorRegistroTest.java`.
3. Ejecuta las pruebas unitarias desde tu IDE.

Los resultados de las pruebas se mostrarán en la ventana de ejecución.
